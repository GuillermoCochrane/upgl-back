Bienvenido a la API de UPGL

Lista de cursos: (/:courseID)

  • Curso de Python: (PYTHON)
  • Curso de IA: (IA)

Endpoints disponibles

  • [ GET ] - /API/course/index : : Información de todos los cursos
  • [ GET ] - /API/course/:courseID : Índice de hipervínculos de la clases del curso seleccionado
  • [ GET ] - /API/course/:courseID/class : Información de todas las clases del curso seleccionado
  • [ GET ] - /API/course/:courseID/class/:classID : Información de una clase del curso seleccionado
  • [ GET ] - /API/course/:courseID/class/:classID/:topicID : Información de un tema de una clase del curso seleccionado
  • [ GET ] - /API/course/:courseID/:indexID : Índice de hipervínculos de una clase del curso seleccionado
  • [ POST ] - /API/course/:courseID/newClass : Endpoint para la creación de una nueva clase del curso seleccionado
  • [ POST ] - /API/course/:courseID/newTopic/:classID : Endpoint para la creación de un nuevo tema de una clase del curso seleccionado
