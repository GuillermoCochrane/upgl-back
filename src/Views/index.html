Bienvenido a la API de UPGL

Endpoints disponibles

Python
  • [ GET ] - /API/python : Índice de hipervínculos de la clases del curso de Python
  • [ GET ] - /API/python/class : Información de todas las clases del curso de Python
  • [ GET ] - /API/python/class/:classID : Información de una clase del curso de Python
  • [ GET ] - /API/python/class/:classID/:topicID : Información de un tema de una clase del curso de Python
  • [ GET ] - /API/python/:indexID : Índice de hipervínculos de una clase del curso de Python
  • [ POST ] - /API/python/newClass : Endpoint para la creación de una nueva clase del curso de Python
  • [ POST ] - /API/python/newTopic/:classID : Endpoint para la creación de un nuevo tema de una clase del curso de Python

  • [ GET ] - /API/ia/class/:classID : Información de una clase del curso de IA
  • [ GET ] - /API/ia/class/:classID/:topicID : Información de un tema de una clase del curso de IA
  • [ GET ] - /API/ia/:indexID : Índice de hipervínculos de una clase del curso de IA
  • [ POST ] - /API/ia/newClass : Endpoint para la creación de una nueva clase del curso de IA
  • [ POST ] - /API/ia/newTopic/:classID : Endpoint para la creación de un nuevo tema de una clase del curso de IA
