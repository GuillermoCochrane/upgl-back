Bienvenido a la API de UPGL

Endpoints disponibles

  • [ GET ] - /API/python: Índice de hipervínculos de la clases del curso de Python
  • [ GET ] - /API/python/class: Información de todas las clases del curso de Python
  • [ GET ] - /API/python/class/:classID: Información de una clase del curso de Python
  • [ GET ] - /API/python/class/:classID/:topicID: Información de un tema de una clase del curso de Python
